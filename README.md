# software-as-service-html

A modern, responsive landing page for an e-commerce solution provider, "min11benja". This project is designed to showcase services for building online stores, including custom themes, app cloning, and digital marketing.

## About The Project

"min11benja" is a static HTML landing page built to convert visitors into clients for e-commerce development services. It features a clean, professional design with sections for services, pricing, and a multi-step contact form.

### Built With

*   [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
*   [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
*   [Tailwind CSS](https://tailwindcss.com)
*   [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Features

*   **Responsive Design**: Fully responsive layout optimized for desktop, tablet, and mobile devices.
*   **Hero Section**: Engaging hero area with call-to-action buttons.
*   **Service Showcases**: Detailed sections for "Selling" (Vende) and "Growing" (Crece) your business.
*   **Step-by-Step Process**: A visual timeline explaining the 6-step process from planning to support.
*   **Pricing Tables**: Clear pricing tiers for different business needs (Freelancer, Startup, Enterprise).
*   **Interactive Contact Form**: A multi-step form to capture client requirements and contact details.
*   **Carousel & Animations**: Smooth visual transitions and carousels to highlight features.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

*   A modern web browser (Chrome, Firefox, Safari, Edge).

### Installation

1.  Clone the repository:
    ```sh
    git clone https://github.com/Min11Benja/software-as-service-html.git
    ```

### Usage

1.  Navigate to the project folder.
2.  Open `index.html` in your web browser.

## Project Structure

*   `index.html`: The main landing page file.
*   `src/`: Source files (Tailwind input CSS).
*   `build/`: Compiled assets including:
    *   `css/`: Compiled stylesheets (`output.css`, etc.)
    *   `js/`: JavaScript files for interactivity.
    *   `img/`: Impactful images for the landing page.
    *   `fonts/`: Custom fonts used in the design.
*   `thankyou.html`: Success page after form submission.
